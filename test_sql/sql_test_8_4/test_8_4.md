## 1.过滤数据
- **关键字：**<font color=Indigo>WHERE</font>
- **语法：**
    ```sql
    SELECT 字段 FROM 表名
    WHERE 过滤条件;
    ```

- **eg:** 

  ```sql
  SELECT * FROM employees
  WHERE last_name  ="King";
  ```
